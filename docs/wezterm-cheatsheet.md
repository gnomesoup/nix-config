# WezTerm Leader Cheat Sheet

Your WezTerm config is managed in `users/modules/wezterm.nix`.

The tab bar is rendered at the bottom of the window.

## Leader

The leader key is `Ctrl-a`.

After pressing `Ctrl-a`, use one of the bindings below before the leader timeout expires.

## Panes

| Keys | Description |
| --- | --- |
| `Ctrl-a -` | Split the current pane vertically |
| `Ctrl-a =` | Split the current pane horizontally |
| `Super-w` | Close the current pane |
| `Ctrl-a m` | Focus pane left |
| `Ctrl-a n` | Focus pane down |
| `Ctrl-a e` | Focus pane up |
| `Ctrl-a i` | Focus pane right |
| `Ctrl-a p` | Enter resize mode |
| `Ctrl-a Shift-m` | Toggle pane zoom |
| `Ctrl-a w` | Open pane selector |

## Resize Mode

Press `Ctrl-a p` and then use the Colemak-DH direction cluster to resize the active pane.

| Keys | Description |
| --- | --- |
| `m` | Resize left |
| `n` | Resize down |
| `e` | Resize up |
| `i` | Resize right |
| `Escape` / `q` | Exit resize mode |

## Modes

| Keys | Description |
| --- | --- |
| `Ctrl-a Enter` | Activate copy mode |

## Copy Mode

| Keys | Description |
| --- | --- |
| `Escape` / `q` / `Ctrl-c` / `Ctrl-g` | Exit copy mode |
| `y` | Copy selection and exit copy mode |
| `v` / `Shift-v` / `Ctrl-v` | Start cell / line / block selection |
| `m` | Move left |
| `n` | Move down |
| `e` | Move up |
| `i` | Move right |
| `Left` / `Down` / `Up` / `Right` | Arrow-key movement |
| `f` / `F` | Move to end of the next word |
| `t` / `T` | Jump forward / backward to a character |
| `r` / `R` | Jump forward / backward until just before a character |

Copy mode also keeps the rest of WezTerm's default bindings, including arrow keys, `PageUp`/`PageDown`, `0`, `$`, `g`, `G`, `o`, search-related motions, and the standard Vim-style navigation keys where they do not conflict with your Colemak overrides.

## Search Mode

| Keys | Description |
| --- | --- |
| `k` | Next match |
| `K` | Previous match |

Search mode also keeps the default WezTerm bindings such as `Enter`, `Escape`, `Ctrl-n`, `Ctrl-p`, `PageUp`, and `PageDown`.

## Tab Bar

The left side of the tab bar shows the current mode.

| Label | Meaning |
| --- | --- |
| `NORMAL` | Default input state |
| `LEADER` | Leader key is active |
| `RESIZE` | Resize mode is active |
| `COPY` | Copy mode is active |
| `SEARCH` | Copy-mode search is active |

The mode color is derived from the active WezTerm theme palette, so changing the theme also changes the indicator colors.

## Notes

- The config is generated by home-manager; edit `users/modules/wezterm.nix` and rebuild with your normal home-manager workflow.
- Copy and search mode both inherit WezTerm defaults and then add your Colemak-style aliases on top.
- `Super-w` now closes the current pane; if that removes the last pane/tab, WezTerm will close the containing tab/window as part of its normal pane lifecycle.
- `Ctrl-a p` enters resize mode, where `m/n/e/i` resize left/down/up/right and `Escape` or `q` exits.
- The tab bar status updates continuously so the `LEADER` indicator appears and clears with the leader timeout.
